### Laravel with Bootstrap & React
Follow these steps, and you good to go!

```bash
git clone https://github.com/irsyadadl/libr.git
cd libr
composer install
cp .env.example .env
php artisan key:generate
npm install
npm run dev
```

Create your database, and now you can migrate!
```bash
php artisan migrate
```
